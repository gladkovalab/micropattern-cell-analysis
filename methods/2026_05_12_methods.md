# Methods – Micropattern Cell Analysis
*2026-05-12*

## Image acquisition and loading

Images were loaded using the `nd2` library (v0.10.3) with `xarray` (v2025.9.0) for labelled multidimensional arrays. Analysis pipelines were prototyped and visualized interactively using Marimo (v0.17.8), a reactive Python notebook environment. All analysis was performed in Python 3.12 using packages managed via the Pixi environment manager.

## Template matching

A micropattern template was defined as an SVG file and rasterized to 1326 × 1326 pixels using CairoSVG (v2.8.2) and PyMuPDF (v1.26.4). The template was zero-padded to 2048 × 2048 pixels for frequency-domain matching. The micropattern center was localized in each image by normalized cross-correlation in frequency space using 2D FFTs (NumPy v2.3.3). The 640 nm channel z-sum projection was used as the reference signal. A template matching score (fractional overlap with Otsu-thresholded image) was computed for quality control.

## Cropping, projections, and background subtraction

A 1024 × 1024 pixel region centered on the matched template was extracted for each fluorescence channel and saved as NetCDF4 (v1.7.3) files. The 405 nm (Hoechst) and 640 nm (template) channels were stored as z-sum projections, while the 488 nm (mitochondria, peroxisomes, synthetic cargo) channel was stored as a maximum-intensity projection to reduce noise. Background was estimated from the empty leftmost and rightmost 128-pixel column strips and subtracted. Experimental conditions and well assignments across plates were specified in a structured Excel spreadsheet read using `fastexcel` (v0.18.0) and Polars (v1.36.1).

## Quantification

For quantification, the MaxIP was rescaled to a normalized intensity (0.1–99.9 percentile) before background subtraction. To quantify the directional distribution of mitochondrial signal, a fixed polygonal wedge was defined in the cropped frame with its apex at the bottom extreme of the micropattern and bounded by the upper tangent points of the arch. For each cell, radial distance from the apex was computed and binned into sixty 1 µm bins. The intensity within each radial bin and within the wedge was summed and expressed as a percentage of total. Two regions were defined based on the TRAK1-mediated kinesin and TRAK2-mediated dynein synthetic cargo localization: peripheral (41–56 µm) and centrosomal (18–33 µm). Per-cell mitochondrial fractions in these regions were compiled into CSV files using Polars (v1.36.1).

## Mean projection imaging

To display mitochondrial distribution for all cells within each condition, unscaled background-subtracted MaxIP images were spatially aligned by center-padding to a common bounding box and summed, then divided by the number of input cells to produce a mean projection image per condition. Mean projections were saved as 16-bit TIFF files using scikit-image (v0.25.2).
